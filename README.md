# About Functions
A function is a block of code that performs a specific task.

Functions make code more readable, reusable, and organized.

# About Recursion
Recursion is a programming technique where a function calls itself until a base condition is met.

It is widely used in problems like factorial, Fibonacci series, and finding GCD.
